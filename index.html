S.NO DATE OBJECT METHOD SYNTAX DEFINITION
GET METHOD
01 Date.prototype.getDate new Date() Return the day of the month (1-31) for the specified date according to local time
02 Date.prototype.getDay new Day() Return the day of the week (0-6) for the specified date according to local time
03 Date.prototype.getFullYear getFullYear() Return the year (4 digits for 4-digits years) for the specified date according to local time
04 Date.prototype.getHours getHours() Return the hours (0-23) in the specified date according to local time
05 Date.prototype.getMilliseconds getMilliseconds() Return the milliseconds (0-999) in the specified date according to local time
06 Date.prototype.getMinutes getMinutes() Return the minutes (0-59) in the specified date according to local time
07 Date.prototype.getMonth getMonth() Return the month (0-11) in the specified date according to local time
08 Date.prototype.getSeconds getSeconds() Return the seconds (0-59) in the specified date according to local time
09 Date.prototype.getTime getTime() Return the numeric value of the specified date as the number of millisecond since january 1,1970,00:00:00 UTC.(Negative values are returned for prior times)
10 Date.prototype.getTimezoneoffset() getTimezoneoffset() Returns the time-zone offset in minutes for the current locale.
11 Date.prototype.getYear() getYear() Returns the year (usually 2-3 digits) in the specified date according to local time.use grtFullyear()*instead.
SET METHOD
01 Date.prototype.setDate() setDate() sets the day of the month for a specified date according to local time.
02 Date.prototype.setFullYear() setFullYear() sets the full years (e.g. 4 digits for 4-digits years) for a specified date according to local time.
03 Date.prototype.setHours() setHours() sets the hours for a specified date according to local time.
04 Date.prototype.setMilliseconds() setMilliseconds() sets the millisecond for a specified date according to local time.
05 Date.prototype.setMinutes() setMinutes() sets the minutes for a specified date according to local time.
06 Date.prototype.setMonth() setMonth() sets the month for a specified date according to local time.
07 Date.prototype.setSeconds() setSeconds() sets the seconds for a specified date according to local time.
08 Date.prototype.setTime() setTime() sets the date object to the time represented by a number of millisecond since january1,1970,00:00:00 UTC.use negative numbers for times prior.
09 Date.prototype.setYear() setYear() set the year (usually 2-3 digits) in the specified date according to local time.use grtFullyear()*instead.
CONVERT
01 Date.prototype.DateString() DateString() return the "date" portion of the date as a human-readable string like 'the apr 12 2018'
02 Date.prototype.toISOString() toISOString() converts a date to a string following the ISO 8601 extended format.
03 Date.prototype.toJSON() toJSON() return a string represting the date using toISOString().intended for use by JSON.stringify().
04 Date.prototype.toGMTString() toGMTString() return a string represting the date based on the GMT(UTC) time zone.use toUTCString()*instead
05 Date.prototype.toLocaleDateString() toLocaleDateString() return a string with a locality sebsitive representation of the date portion of this date based on system settings.
06 Date.prototype.toLocaleString() toLocaleString() return a string with a locality-sebsitive representation of the date. overrides the object.prototype.toLocaleString() method
07 Date.prototype.toLocaleTimeString() toLocaleTimeString() return a string with a locality-sebsitive representation of the time portion of this date,based on system settings.
08 Date.prototype.toString() toString() return a string representing the specified date object.overrides the object.prototype.toString()*method
09 Date.prototype.TimeString() TimeString() return the "time" portion of the date as a human-readable string.
